# VoxelForge

A small Metal-based voxel engine prototype for macOS.

## Controls

- `WASD` move
- `Q/E` turn left/right
- `Up/Down` look up/down

## Features

- Chunk-based voxel terrain
- Camera-controlled first-person navigation
- Green voxel rendering
- Metal depth testing

## Notes

This is an early prototype and still uses simple chunk generation and basic meshing.
